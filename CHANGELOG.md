# Changelog

รูปแบบอิงตาม [Keep a Changelog](https://keepachangelog.com/) และเวอร์ชั่นตาม [SemVer](VERSIONING.md)

## [Unreleased]

## [1.2.1] - 2026-06-15

### Fixed
- แก้ขอบการ์ดสว่างผิดปกติใน dark mode — เปลี่ยนเส้นขอบการ์ดจาก `ring-brand-100` (สีพีชอ่อนคงที่) มาใช้สีขอบตามธีม (`ring-border`) ให้มืดกลมกลืนใน dark mode ทั้ง 3 ธีม

## [1.2.0] - 2026-06-15

### Added
- เข้าสู่ระบบด้วย OTP ผ่าน LINE (passwordless) สำหรับบัญชีที่ผูก LINE แล้ว — กรอกอีเมล → รับรหัส 6 หลักทาง LINE → ยืนยัน (OTP เก็บแบบ hash, อายุ 5 นาที, ล็อกหลังใส่ผิด 5 ครั้ง, ขอใหม่ได้ทุก 60 วิ) พร้อมทางสำรอง "ใช้รหัสผ่านแทน"
- หน้าประวัติการเติมวัน (ledger ของวันใช้งานที่ได้รับ) ในหน้าบิล
- หน้า **สถิติร้าน** (/shop/analytics): การ์ดตัวเลข (แสตมป์/แลกรางวัล/ลูกค้าแอคทีฟ/อัตราการแลก/ลูกค้าใหม่) + กราฟแนวโน้มรายวัน (recharts) + แยกตามประเภทแสตมป์/สาขา + ลูกค้าสะสมสูงสุด เลือกช่วง 7/30/90 วัน
- หน้า **สถิติระบบ** (/admin/analytics) สำหรับผู้ดูแล: ภาพรวมข้ามทุกร้าน (แสตมป์/แลกรางวัล/ลูกค้าแอคทีฟ/อัตราการแลก/ลูกค้าใหม่/ลูกค้าทั้งหมด/ร้านทั้งหมด/ร้านที่มีกิจกรรม) + กราฟแนวโน้มรายวันรวม + leaderboard ร้านค้าที่แอคทีฟสูงสุด เลือกช่วง 7/30/90 วัน

### Changed
- รายการที่ยาว (ประวัติแลกรางวัล, การแจ้งเตือน, คิวชำระเงิน, รายชื่อลูกค้า, คำขอติดต่อ) เปลี่ยนเป็นโหลดแบบ cursor "โหลดเพิ่ม" แทนการตัดจบที่จำนวนคงที่
- เก็บงานเล็กๆ: คอมโพเนนต์ `<Textarea>` ร่วม, รวมฟอร์แมตวันที่/ป้ายข้อความให้สม่ำเสมอ, เพิ่ม loading skeleton ที่ยังขาด

## [1.1.0] - 2026-06-15

### Added
- รองรับ **แสตมป์หลายประเภทต่อร้าน** (เช่นแบ่งตามราคา 10/20/30฿) — แต่ละประเภทมีเกณฑ์ครบและของรางวัลของตัวเอง สะสมแยกกัน พนักงานเลือกประเภทตอนกดแสตมป์ (ร้านประเภทเดียวพฤติกรรมเหมือนเดิม)

## [1.0.0] - 2026-06-15

รุ่น production แรก — ระบบบัตรสะสมแสตมป์ดิจิทัลสำหรับร้านค้าหลายสาขา (ปิดเฟส 2)

### Added
- บัตรสะสมแสตมป์ผ่านการสแกน QR สำหรับลูกค้า (ไม่ต้องพกบัตร) และระบบหลายสาขา
- ระบบบิลแบบ "เติมวัน" (day top-up) + แจ้งสลิป PromptPay + แอดมินตรวจอนุมัติ
- แผนที่ร้านค้าหน้าแรก + ปุ่มระบุตำแหน่งปัจจุบัน (ลูกค้า) และปักหมุดสาขา (ร้านค้า)
- ประวัติการแลกของรางวัล (ฝั่งร้านค้าและลูกค้า)
- ระบบแจ้งเตือนในแอป (กระดิ่ง + หน้า inbox) และ LINE OA push สำหรับ admin/เจ้าของร้าน
- ช่องทางร้านค้าติดต่อแอดมิน พร้อมสถานะคำขอและการแจ้งเตือนกลับเมื่อแอดมินรับเรื่อง
- แสดงเลขเวอร์ชั่น + commit ที่ท้ายทุกหน้า

### Changed
- เปลี่ยนเมนูฝั่งร้านค้า/แอดมินเป็น bottom tab bar + เมนู "เพิ่มเติม" ทุกขนาดจอ
- ราคาแพ็กเกจเติมวันสำเร็จรูปคิดจาก "ราคา/วัน" ของแต่ละร้าน (แทนราคาตายตัว)
- เพิ่ม loading skeleton ทุกหน้าให้รู้สึกลื่นขึ้นระหว่างโหลด
