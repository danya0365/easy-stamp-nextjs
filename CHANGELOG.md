# Changelog

บันทึกการเปลี่ยนแปลงที่สำคัญของ Easy Stamp
รูปแบบอิงตาม [Keep a Changelog](https://keepachangelog.com/) และเวอร์ชั่นตาม [SemVer](VERSIONING.md)

## [Unreleased]

## [1.5.0] - 2026-06-16

### Added
- **ปิดร้านชั่วคราว (pause)** — เจ้าของร้าน (และแอดมิน) ปิดร้านชั่วคราวได้เมื่อไม่เปิดบริการ (วันหยุดยาว/ปรับปรุงร้าน) โดย **ระบบหยุดนับวันใช้งาน วันที่เหลือไม่ถูกหัก** พอเปิดอีกครั้งวันคงเหลือเท่าเดิม · ระหว่างปิด: กดแสตมป์/แลกรางวัลไม่ได้ ร้านถูกซ่อนจากแผนที่ พนักงานเข้าใช้งานไม่ได้ และบัตรลูกค้าแสดงป้าย "ปิดชั่วคราว" (ดูแต้มได้) · เจ้าของยังเข้าระบบมากด "เปิดร้านอีกครั้ง" ได้ตลอด

## [1.4.1] - 2026-06-16

### Fixed
- แก้ dark mode อ่านยาก/สีเพี้ยนทุกธีม (minimal, retro, cafe) — เดิม dark override สีแบรนด์ไม่ครบ ทำให้ตัวอักษร (ลิงก์/ปุ่ม), ขอบปุ่ม และกล่องสถานะว่าง (EmptyState) แสดงผลผิด; เพิ่มสีแบรนด์ครบทุกเฉดใน dark + แยกสีตัวอักษรบนปุ่ม (`--on-brand`) ให้คอนทราสต์อ่านออกทุกธีม

## [1.4.0] - 2026-06-16

### Added
- **เข้าสู่ระบบด้วย OTP เท่านั้นสำหรับบัญชีที่ผูก LINE** — เจ้าของร้าน/พนักงานที่ผูก LINE แล้วต้องเข้าระบบผ่านรหัส OTP ทาง LINE (ปิดการใช้รหัสผ่านเพื่อความปลอดภัย); ผู้ดูแลระบบยังใช้รหัสผ่านได้เป็นทางสำรอง
- **แจ้งเตือนทาง LINE เมื่อเข้าสู่ระบบสำเร็จ** ทุกครั้ง (บอกเวลา/วิธี — หากไม่ใช่เจ้าของบัญชีจะรู้ตัวทันที)
- **ปุ่ม "ติดต่อผู้ดูแล" บนหน้าเข้าสู่ระบบ** สำหรับผู้ที่เข้าระบบไม่ได้ — ส่งเรื่องถึงผู้ดูแลทันที (แจ้งในแอป + LINE) เพื่อให้รีบช่วยเหลือ
- ป้องกันสแปม/บอทบนฟอร์มติดต่อสาธารณะ: CAPTCHA (Cloudflare Turnstile, เปิดเมื่อตั้งค่า key) + honeypot + จำกัดจำนวนครั้งต่อ IP

### Changed
- เพิ่มการจำกัดอัตราการเรียก (rate-limit) ต่อ IP สำหรับการขอ OTP และการล็อกอินด้วยรหัสผ่าน (กัน OTP-bombing / เดารหัสผ่าน); หน้าเข้าสู่ระบบตั้ง noindex

## [1.3.0] - 2026-06-16

### Added
- **เข้าสู่ระบบจากบัญชีที่เคยใช้บนเครื่องนี้** (แบบ Facebook) — หน้า login จำบัญชีที่เคยเข้าสำเร็จ แตะเลือกได้เลยไม่ต้องพิมพ์อีเมล (ยังต้องยืนยันตัวตนด้วย OTP/รหัสผ่านตามเดิม; เก็บแค่อีเมล+บทบาทในคุกกี้ httpOnly ไม่เก็บรหัสผ่าน/เซสชัน) พร้อมปุ่มลบบัญชีออกจากเครื่อง
- ปุ่มหน้าแรกรู้สถานะการเข้าสู่ระบบ: ถ้าเข้าระบบอยู่แล้วลิงก์ตรงไปแดชบอร์ด (ไม่วนผ่านหน้า login) + แสดงสถานะกำลังโหลดทันทีที่กด + เพิ่ม loading ให้หน้า login
- **พนักงานสาขา (staff) ใช้งานครบเทียบเท่าร้าน/แอดมิน** — เพิ่มแถบเมนูล่าง, หน้าตั้งค่า, หน้าแจ้งเตือน (+กระดิ่ง) และเชื่อมต่อ LINE ได้ (เดิมพนักงานเชื่อม LINE ไม่ได้)
- โหมดนักพัฒนา (เฉพาะเครื่อง local): สลับเข้าสู่ระบบเป็นผู้ใช้ใดก็ได้แบบคลิกเดียวเพื่อทดสอบ (กันหลุด production สองชั้น)

### Changed
- ปรับส่วนเชื่อมต่อ LINE เป็น **"ช่องทางเชื่อมต่อ & ความปลอดภัย"** สื่อชัดว่าใช้ทั้งรับแจ้งเตือนและเข้าสู่ระบบด้วย OTP (วางโครงรองรับช่องทางอื่นในอนาคต) ครบทุกบทบาท
- เปลี่ยนลูกศรข้อความ ←/→ เป็นไอคอน (lucide) ให้คมและสม่ำเสมอทุกหน้า
- ธีม retro: หัวข้อใช้ฟอนต์ Noto Serif Thai (อารมณ์นิตยสาร รองรับภาษาไทย) — เดิมอ้างฟอนต์ที่ไม่มีตัวไทยและไม่ถูกโหลด

## [1.2.1] - 2026-06-15

### Fixed
- แก้ขอบการ์ดสว่างผิดปกติใน dark mode — เปลี่ยนเส้นขอบการ์ดจาก `ring-brand-100` (สีพีชอ่อนคงที่) มาใช้สีขอบตามธีม (`ring-border`) ให้มืดกลมกลืนใน dark mode ทั้ง 3 ธีม

## [1.2.0] - 2026-06-15

### Added
- เข้าสู่ระบบด้วย OTP ผ่าน LINE (passwordless) สำหรับบัญชีที่ผูก LINE แล้ว — กรอกอีเมล → รับรหัส 6 หลักทาง LINE → ยืนยัน (OTP เก็บแบบ hash, อายุ 5 นาที, ล็อกหลังใส่ผิด 5 ครั้ง, ขอใหม่ได้ทุก 60 วิ) พร้อมทางสำรอง "ใช้รหัสผ่านแทน"
- หน้าประวัติการเติมวัน (ledger ของวันใช้งานที่ได้รับ) ในหน้าบิล
- หน้า **สถิติร้าน** (/shop/analytics): การ์ดตัวเลข (แสตมป์/แลกรางวัล/ลูกค้าแอคทีฟ/อัตราการแลก/ลูกค้าใหม่) + กราฟแนวโน้มรายวัน (recharts) + แยกตามประเภทแสตมป์/สาขา + ลูกค้าสะสมสูงสุด เลือกช่วง 7/30/90 วัน
- หน้า **สถิติระบบ** (/admin/analytics) สำหรับผู้ดูแล: ภาพรวมข้ามทุกร้าน (แสตมป์/แลกรางวัล/ลูกค้าแอคทีฟ/อัตราการแลก/ลูกค้าใหม่/ลูกค้าทั้งหมด/ร้านทั้งหมด/ร้านที่มีกิจกรรม) + กราฟแนวโน้มรายวันรวม + leaderboard ร้านค้าที่แอคทีฟสูงสุด เลือกช่วง 7/30/90 วัน

### Changed
- รายการที่ยาว (ประวัติแลกรางวัล, การแจ้งเตือน, คิวชำระเงิน, รายชื่อลูกค้า, คำขอติดต่อ) เปลี่ยนเป็นโหลดแบบ cursor "โหลดเพิ่ม" แทนการตัดจบที่จำนวนคงที่
- เก็บงานเล็กๆ: คอมโพเนนต์ `<Textarea>` ร่วม, รวมฟอร์แมตวันที่/ป้ายข้อความให้สม่ำเสมอ, เพิ่ม loading skeleton ที่ยังขาด

## [1.1.0] - 2026-06-15

### Added
- รองรับ **แสตมป์หลายประเภทต่อร้าน** (เช่นแบ่งตามราคา 10/20/30฿) — แต่ละประเภทมีเกณฑ์ครบและของรางวัลของตัวเอง สะสมแยกกัน พนักงานเลือกประเภทตอนกดแสตมป์ (ร้านประเภทเดียวพฤติกรรมเหมือนเดิม)

## [1.0.0] - 2026-06-15

รุ่น production แรก — ระบบบัตรสะสมแสตมป์ดิจิทัลสำหรับร้านค้าหลายสาขา (ปิดเฟส 2)

### Added
- บัตรสะสมแสตมป์ผ่านการสแกน QR สำหรับลูกค้า (ไม่ต้องพกบัตร) และระบบหลายสาขา
- ระบบบิลแบบ "เติมวัน" (day top-up) + แจ้งสลิป PromptPay + แอดมินตรวจอนุมัติ
- แผนที่ร้านค้าหน้าแรก + ปุ่มระบุตำแหน่งปัจจุบัน (ลูกค้า) และปักหมุดสาขา (ร้านค้า)
- ประวัติการแลกของรางวัล (ฝั่งร้านค้าและลูกค้า)
- ระบบแจ้งเตือนในแอป (กระดิ่ง + หน้า inbox) และ LINE OA push สำหรับ admin/เจ้าของร้าน
- ช่องทางร้านค้าติดต่อแอดมิน พร้อมสถานะคำขอและการแจ้งเตือนกลับเมื่อแอดมินรับเรื่อง
- แสดงเลขเวอร์ชั่น + commit ที่ท้ายทุกหน้า

### Changed
- เปลี่ยนเมนูฝั่งร้านค้า/แอดมินเป็น bottom tab bar + เมนู "เพิ่มเติม" ทุกขนาดจอ
- ราคาแพ็กเกจเติมวันสำเร็จรูปคิดจาก "ราคา/วัน" ของแต่ละร้าน (แทนราคาตายตัว)
- เพิ่ม loading skeleton ทุกหน้าให้รู้สึกลื่นขึ้นระหว่างโหลด
